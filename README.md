# E-commerce Sales Analytics Dashboard
E-commerce sales data analysis using Python and Power BI with business insights.

## Project Overview
This project analyzes e-commerce transaction data to identify revenue trends, customer behavior, and regional sales performance.

## Tools Used
- Python (Pandas, NumPy)
- Power BI
- Data Visualization

## Key Analysis
• Revenue trend analysis
• Regional sales performance
• Order status distribution
• Customer value analysis
• Delivery time analysis

## Dashboard Preview
![Dashboard](images/dashboard.png)

## Key Insights
- Revenue shows steady growth over time.
- A small number of regions generate the majority of orders.
- High-value customers contribute a significant share of revenue.
- Delivery time varies across regions.
